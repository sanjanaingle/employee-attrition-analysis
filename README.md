# Employee Attrition Analysis

## Overview
This project analyzes employee attrition data to understand why employees leave an organization. The analysis focuses on identifying patterns related to department, salary, age, and job satisfaction.

## Objectives
- Calculate overall attrition rate
- Analyze attrition by department
- Study the relationship between salary and attrition
- Examine attrition across age groups
- Evaluate the impact of job satisfaction on employee retention

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- CSV Dataset

## Key Analysis
- Employee Attrition Rate
- Department-wise Attrition
- Age Group Analysis
- Salary Comparison
- Job Satisfaction Analysis
- Data Visualization

## Key Insights
- Identified departments with the highest employee turnover.
- Compared average salaries of employees who left versus those who stayed.
- Analyzed age groups most affected by attrition.
- Evaluated the correlation between job satisfaction and employee retention.
